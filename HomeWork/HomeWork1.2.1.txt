Domain Name System- распределительная система для получения информации о доменах или другими словами это технология, котрая позволяет через браузер найти запрошенный пользователем сайт по его имяни. Домен или доменное имя- уникальное имя оттдалённого Веб-сервера распологающегося по IP-адресу.
Например:
В DNS регестрируется, что по такому IP-адресу  находится конкретое имя, например google.com. По этому IP- адресу можт размещаться Веб-сервер, на котором может находится  веб-приложение (сайт). Доступ к этому сайту может осуществляться через доменное имя  google.com.
HTTP (HyperText Transfer Protocol) - Протакол передачи данных, который подразумевает набор операций, запрос -> ответ. 
HTTPS (HyperText Transfer Protocol Secure)- протокол передачи данный при помощи ключей шифрования, осуществяется с помощью сертификата- SSL. Существует публичный и приватный ключь. Публичный ключь набор символов, отправлямый вместе с запросом браузера к серверу, он доступен всем. Привятный ключь известен только владельцу сайта, используется для расшифровки данных, отправленных браузером.
